# **App Name**: HumanizeAI Pro

## Core Features:

- Undetectable Mode: Aggressive pattern removal to bypass AI detection tools like Turnitin, ZeroGPT, and GPTZero.
- Contextual Rephraser: Uses deep semantic analysis to ensure the original meaning and intent of the message remains intact.
- Multilingual Sidebar: Allows users to switch the UI and output language (Arabic, English, Spanish, French, German, Chinese, etc.).
- Tone Selector: Provides options to choose the output tone (Professional, Academic, Casual, Creative).
- One-Click Humanizer: A prominent button to trigger the text transformation process.
- Semantic Header Tags: Auto-generates H1, H2, and H3 tags within the rephrased output.
- LSI Keyword Integration: Automatically injects related semantic keywords to boost ranking without keyword stuffing; acts as a tool.
- Meta-Tag Generator: Generates SEO Meta Titles and Descriptions for the rephrased content.

## Style Guidelines:

- Use Vibrant Mesh Gradients combining Red (#FF416C), Blue (#00B4DB), Green (#00F260), and Yellow (#F7971E).
- Modern Glassmorphism with semi-transparent cards, subtle borders, and Backdrop Blur.
- A clean white/light-gray base with moving, soft gradient orbs in the corners using the brand colors.
- Headline font: 'Space Grotesk' for a tech feel.
- Body font: 'Inter' for a human feel.
